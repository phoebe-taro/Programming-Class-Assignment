# 程式設計課程作業 (Programming-Class-Assignment)

這是我的課程學習紀錄，主要使用 Python 與 Google Colab 進行實作。

* **Ch2 - 資料型別與運算子**：變數處理、型別轉換、進位制轉換

* **Ch6 - 函數 (Functions)**：參數傳遞機制、遞迴、Lambda 表達式
* **Ch7 - 物件導向 (OOP)**：類別 (Class)、繼承、實例與靜態函數
* **Ch8 - 檔案與異常處理**：File I/O、Try-Except 異常捕捉、模組封裝
* **Ch9~10 - Numpy 矩陣運算**
* **Ch11 - Matplotlib 資料視覺化**
